# COMMAND LIST (Bot)

## File generate (tạo skeleton commands/events/scripts)

Trong thư mục `bot/`:

```bash
cd bot
node gen.js
```

---

## NPM

```bash
cd bot
npm install
```

---

## Chạy Bot

```bash
cd bot
npm run deploy   # Đăng ký slash commands lên Discord
npm start        # Chạy bot (production)
npm run dev      # Chạy bot với nodemon (development)
```
