# Contributing to Discord Counselor

Thank you for your interest in contributing to the Discord Counselor project! We welcome all contributions, from bug reports to new features.

## How to Contribute

### 1. Reporting Bugs
- Use the project's Issue Tracker.
- Provide a clear and descriptive title.
- Detail the steps to reproduce the issue.

### 2. Suggesting Features
- Open an Issue with the tag `enhancement`.
- Describe the feature and why it would be useful.

### 3. Pull Requests
- Fork the repository.
- Create a new branch (e.g., `feature/awesome-feature`).
- Make your changes.
- Ensure your code follows the established style (see `.editorconfig`).
- Submit a Pull Request with a clear description of the changes.

## Development Setup
Refer to the `README.md` for instructions on setting up the backend, directive, and frontend components.

## Code of Conduct
Please be respectful and professional in all interactions within the project community.
