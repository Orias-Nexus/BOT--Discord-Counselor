# Infra

Cấu hình hạ tầng: Terraform, Kubernetes, scripts cài đặt server.

## Nội dung (placeholder)

- **Terraform**: `*.tf` cho cloud (AWS/GCP/Render) khi cần.
- **Kubernetes**: `k8s/*.yaml` nếu chạy trên cluster.
- **Scripts**: `scripts/` — cài đặt Node, env, systemd cho Bot/Backend trên VPS.

Thêm file khi bắt đầu deploy production.
